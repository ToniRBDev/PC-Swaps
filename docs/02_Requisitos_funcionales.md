- Crear cuenta.
- Eliminar cuenta.
- Iniciar sesión.
- Cerrar sesión.
- Modificar datos del usuario.
- Modificar foto de perfil.
- Modificar contraseña.

## Productos

- Publicar producto.
- Eliminar producto publicado.
- Ver productos publicados.
- Editar producto publicado.
- Renovar producto publicado
- Ver productos publicados por otros usuarios.
- Ver detalles de un producto publicado por otro usuario
- Ver datos de contacto del vendedor.
- Buscar productos por categoría.
- Buscar productos mediante barra de búsqueda.

## Seguimiento

- Añadir producto a seguimiento.

- Eliminar conversación.
